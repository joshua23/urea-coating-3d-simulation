# 产线动态流程模拟系统

这是一个使用HTML、CSS和JavaScript创建的产线动态流程模拟系统，用于可视化显示各主要生产设备的运行/停止状态。

## 主要功能

- 动态显示各设备（投料仓、提升机、筛分机、抛光机、包膜机、冷却塔、包装机）的运行状态
- 为每种设备提供独特的动画效果和精确的3D模型
- 支持自动模式，顺序模拟设备状态变化
- 提供设备详细信息查看功能
- 支持手动控制单个设备或全部设备的状态
- **新增功能**：
  - 实时监控面板：显示各设备当前运行状态
  - 参数调节：可调整各设备的运行参数（如速度、频率等）
  - 故障模拟：可模拟各种设备故障情况，包含多种故障类型的视觉效果

## 使用方法

1. 下载项目文件
2. 在浏览器中打开`index.html`文件
3. 使用界面上的按钮控制设备状态：
   - 点击"启动所有设备"按钮启动所有设备
   - 点击"停止所有设备"按钮停止所有设备
   - 点击"自动模式"按钮开启自动模拟
4. **新增操作**：
   - 使用实时监控面板中的滑块调整设备参数
   - 通过故障模拟面板选择设备和故障类型，点击"触发故障"模拟故障情况

## 交互方式

- 单击设备：查看设备的详细信息，可切换设备运行/停止状态
- 旋转视图：通过"旋转视图"按钮360度查看生产线
- 重置视图：通过"重置视图"按钮返回初始视角
- **新增交互**：
  - 参数调节：通过滑块实时调整设备运行参数，动态更改动画效果
  - 故障模拟：选择设备和故障类型，观察故障状态下的视觉效果

## 项目文件结构

- `index.html` - 主HTML文件
- `styles.css` - CSS样式文件
- `script.js` - JavaScript脚本文件

## 技术特点

- 纯前端实现，无需服务器
- 响应式设计，适配不同屏幕尺寸
- 使用CSS动画提供流畅的视觉效果
- 模块化JavaScript代码，易于维护和扩展
- **新增特性**：
  - 设备参数动态调整：动态变更CSS动画时间实现运行速度调整
  - 故障状态可视化：应用特殊动画和滤镜效果模拟不同类型故障
  - 监控面板实时同步：设备状态变更时即时更新监控面板

## 设备规格参数

- **投料机**：电机械振动式，容量42-280吨/小时，流速40 ft/min
- **Z型提升机**：卸料高度3296mm，链条节距38.1mm，运行速度9-11 m/min
- **液压分批筒**：内径50mm，杆径30mm，行程200mm，循环时间8秒
- **包膜机**：直径2.44米，旋转速度8-10 RPM
- **输送机系统**：宽度650mm，多段配置（2×20米，2×14米）
- **提升装置**：最高提升高度17米
- **冷却装置**：立窗式，空气流动冷却
