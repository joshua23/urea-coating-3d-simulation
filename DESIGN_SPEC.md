# 设计规范文档（Design Specification Document）

## 1. 设计概述

### 1.1 设计理念
**核心理念**：构建一个智能化、可视化、高效率的工业4.0生产线监控系统，通过深色主题设计降低操作员视觉疲劳，通过3D可视化技术直观展示生产流程，通过实时数据监控确保生产安全高效。

### 1.2 目标用户
- **主要用户**：生产线操作员、工艺工程师、生产主管
- **使用场景**：24/7连续监控、故障诊断、参数调整、生产报告
- **用户特征**：技术背景强、需要长时间监控、对数据准确性要求高

### 1.3 设计原则
1. **清晰性原则**：信息层级分明，关键数据一目了然
2. **效率性原则**：减少操作步骤，提供快捷操作方式
3. **一致性原则**：统一的视觉语言和交互模式
4. **可靠性原则**：故障状态醒目，确保安全生产
5. **可访问性原则**：支持色盲模式，提供键盘导航

### 1.4 设计目标
- 降低操作员认知负荷，提升监控效率30%
- 减少故障响应时间至3秒内
- 支持同时监控9个设备的运行状态
- 实现数据更新延迟低于100ms
- 适配1080p至4K多种显示设备

### 1.5 品牌定位
专业、可靠、创新、智能的工业监控解决方案，体现技术领先性和工业美学。

## 2. 视觉设计规范

### 2.1 色彩系统

#### 主色调
- **深空蓝（主背景）**：`#0A0E27` - 主界面背景，营造专业技术氛围
- **午夜蓝（次背景）**：`#151B3D` - 面板背景，创建视觉层次
- **科技蓝（品牌色）**：`#00D4FF` - 活跃元素、选中状态、数据高亮

#### 功能色彩
- **运行绿**：`#00FF88` - 设备正常运行、成功操作反馈
- **警告黄**：`#FFB800` - 警告信息、需要注意的参数
- **故障红**：`#FF3366` - 故障状态、紧急停止、危险操作
- **信息蓝**：`#3B82F6` - 一般信息提示、链接、辅助信息
- **待机灰**：`#6B7280` - 停止状态、禁用元素、非活跃内容

#### 中性色系统
- **高亮白**：`#FFFFFF` - 重要文字、活跃状态
- **主文字**：`#E5E7EB` - 主要文字内容
- **次文字**：`#9CA3AF` - 次要文字、说明文字
- **边框色**：`#374151` - 分割线、边框
- **深边框**：`#1F2937` - 深色分割线
- **卡片背景**：`#1A1F3A` - 卡片、浮层背景
- **悬停背景**：`#2A3050` - 悬停状态背景

#### 渐变色彩
- **数据渐变**：`linear-gradient(135deg, #667EEA 0%, #764BA2 100%)` - 数据可视化
- **状态渐变**：`linear-gradient(90deg, #00D4FF 0%, #00FF88 100%)` - 正常运行状态
- **警告渐变**：`linear-gradient(90deg, #FFB800 0%, #FF6B6B 100%)` - 警告到危险过渡

#### 色彩使用规范
- **背景层级**：深空蓝(基底) → 午夜蓝(面板) → 卡片背景(组件)
- **文字对比度**：确保所有文字与背景对比度≥7:1（WCAG AAA级）
- **状态指示**：使用高饱和度色彩，确保在深色背景下清晰可见
- **数据可视化**：使用渐变增强视觉吸引力，但保持数据清晰度

## 3. 交互设计规范

### 3.1 导航系统

#### 主控制栏
- **位置**：顶部固定，高度80px
- **内容**：系统标题 + 主要控制按钮 + 视图控制 + 系统状态
- **交互**：按钮hover发光效果，点击缩放反馈
- **响应式**：移动端收缩为汉堡菜单

#### 设备导航
- **3D场景导航**：鼠标滚轮缩放，右键拖拽旋转，左键点击选择
- **键盘导航**：Tab键切换设备，Enter确认，Esc取消
- **触摸导航**：双指缩放，单指旋转，点击选择

#### 快速跳转
- **设备快捷键**：数字键1-9对应9个设备
- **视图快捷键**：V键切换视图，R键重置视图
- **控制快捷键**：空格键暂停/启动，S键停止所有

### 3.2 交互反馈

#### 按钮状态
```css
/* 默认状态 */
.button {
  background: linear-gradient(135deg, #1A1F3A, #2A3050);
  border: 1px solid #374151;
```

#### 设备交互状态
- **默认状态**：半透明边框，基础光照
- **悬停状态**：发光边框，亮度提升，显示名称标签
- **选中状态**：强发光效果，显示信息面板
- **运行状态**：绿色呼吸灯效果，动画运转
- **故障状态**：红色闪烁警告，震动动画

#### 表单交互
- **输入框聚焦**：边框发光，背景轻微变亮
- **滑块调节**：实时数值显示，平滑过渡动画
- **下拉选择**：展开动画，选项hover高亮
- **开关切换**：滑动动画，颜色渐变过渡

### 3.3 系统反馈

#### 通知系统
- **成功通知**：顶部滑入，绿色背景，3秒后自动消失
- **警告通知**：右上角弹出，黄色边框，需手动关闭
- **错误通知**：居中模态框，红色强调，必须确认
- **信息通知**：右下角提示，蓝色图标，5秒后淡出

#### 加载状态
- **全局加载**：全屏半透明遮罩 + 品牌Logo动画
- **局部加载**：区域内骨架屏 + 脉冲动画
- **数据加载**：进度条 + 百分比显示
- **3D加载**：模型线框 → 材质 → 完整渲染

### 3.4 动效规范

#### 时间曲线
```css
/* 标准缓动 */
--ease-standard: cubic-bezier(0.4, 0, 0.2, 1);

/* 减速缓动 - 进入动画 */
--ease-decelerate: cubic-bezier(0, 0, 0.2, 1);

/* 加速缓动 - 退出动画 */
--ease-accelerate: cubic-bezier(0.4, 0, 1, 1);

/* 弹性缓动 - 强调动画 */
--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

#### 动画时长
- **微交互**：100-200ms（按钮、开关）
- **状态转换**：200-300ms（卡片展开、面板切换）
- **页面转场**：300-400ms（视图切换、弹窗）
- **复杂动画**：400-600ms（3D旋转、数据加载）

#### 3D动画
- **设备旋转**：Y轴360度，持续20秒，线性
- **呼吸灯效**：透明度0.5-1，持续2秒，ease-in-out
- **故障震动**：X轴±5px，持续0.1秒，10次循环
- **流体动画**：贝塞尔路径，表示物料流动
